# Портфолио `derfex`

## TypeScript

Основной репозиторий TypeScript проектов: [elaboration][elaboration].
Там сейчас Angular, Vue, Nest, Nx.

### Pet shop / *Зоомагазин*

Список товаров с возможностями фильтрации, сортировки и добавления в корзину.

- [развёрнутое приложение][pet-shop-deployed]
- [исходный код][pet-shop-source]

Технологии:
Angular 20, Angular Material, Nx, ESLint, Prettier.

### Philistine calculations / *Обывательские расчёты*

Обывательские калькуляторы, которыми пользуюсь в повседневной жизни.

- [развёрнутое приложение][philistine-calculations-deployed]
- [исходный код][philistine-calculations-source]

Технологии:
Vue 3 (composition API), `vue-router`, SASS, Nx, ESLint, Prettier.

### Vehicles registry / *Реестр транспорта*

Реестр транспорта с возможностями создавать, просматривать, изменять и удалять (C. R. U. D.) записи.

- [развёрнутое приложение][vehicles-registry-deployed]
- [исходный код][vehicles-registry-source]

Технологии:
Vue 3 (composition API), Pinia, `vue-router`, Vuetify, SASS, Nx, ESLint, Prettier.

## Нативное, чистое, vanilla

Есть коллекция проектов на чистых HTML, CSS, JS (ES5+), SVG: [pure-tasks][pure-tasks].

### Tic-Tac-Toe / *Крестики-Нолики*

Популярная игра для двоих с реализованным и оптимизированным алгоритмом определения победителя.

- [deployed application][tic-tac-toe-deployed]
- [source code][tic-tac-toe-source]

Технологии (как и обещал):
HTML, CSS, JS.
